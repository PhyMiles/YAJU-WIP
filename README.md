iPhone / iPad (iOS 系統)

請務必使用內建的 Safari 瀏覽器來操作：

打開網頁： 在 Safari 瀏覽器中貼上並打開你的專屬網址。

尋找分享按鈕： 點擊螢幕正下方工具列正中央的**「分享」圖示**（一個方框裡面有一個朝上的箭頭）。

加入主畫面： 在彈出的選單中往下滑，找到並點選 「加入主畫面」 (Add to Home Screen) 的選項。

確認新增： 畫面上會出現 App 的名稱，你可以修改它（例如改成「進度排程」），然後點擊右上角的 「新增」。




 Android 手機 (三星、Sony、Google 等)

請務必使用 Google Chrome 瀏覽器來操作：

打開網頁： 在 Chrome 瀏覽器中貼上並打開你的專屬網址。

開啟選單： 點擊螢幕右上角的 「三個點」 (⋮) 圖示來打開選單。

加到主畫面： 在選單中尋找並點選 「加到主畫面」 (Add to Home screen) 或是「安裝應用程式」。

確認新增： 系統會跳出確認視窗，點擊 「新增」 或「加入」。
